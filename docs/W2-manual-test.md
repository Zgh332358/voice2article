# W2 浏览器手测清单

W2 交付物（语音输入 → 转写 → 入库 → 多会话切换）的人工验证清单。
跑完前面 `scripts/w2-smoke.sh`（端到端绿）之后，再照这份在浏览器里走一遍。

## 启动

两个终端：

```bash
# 终端 A
cd ~/Documents/voice2article/backend && uv run uvicorn app.main:app --reload

# 终端 B
cd ~/Documents/voice2article/frontend && pnpm dev
```

打开 [http://localhost:5173](http://localhost:5173)。

## 1. 鉴权与路由

| # | 操作 | 预期 |
|---|---|---|
| 1.1 | 无登录态访问 `/` | 自动跳 `/login` |
| 1.2 | 无登录态访问 `/conversations` | 跳 `/login`，state.from 应记得回去 |
| 1.3 | 注册新邮箱（≥8 位密码） | 201 + 自动登录 + 跳到来源页 |
| 1.4 | 退出登录 | 跳回 `/login`，localStorage 清空 |
| 1.5 | 错误密码登录 | 表单内红色 Alert "邮箱或密码错误"，**不重复弹 toast** |
| 1.6 | 未知路由（`/zzz` 等） | 登录态显示 404 页面（带回首页按钮）|

## 2. 录音 → 转写

| # | 操作 | 预期 |
|---|---|---|
| 2.1 | 第一次点"开始录音" | 浏览器请求麦克风权限 |
| 2.2 | 拒绝权限 | 红色 Alert "麦克风权限被拒绝：…" |
| 2.3 | 同意权限后说一句话（5~10 秒） | 计时器 mm:ss 实时跳；显示编码（webm/mp4 因浏览器而异）|
| 2.4 | 点"停止并转写" | 出现 "正在调用 Step ASR 转写…"；4 秒内中文带标点的文本出现在右侧消息气泡 |
| 2.5 | 网络断开后再录 | 错误提示明确（"请求失败" / 后端不可达），UI 不卡死 |

## 3. 文件上传

| # | 操作 | 预期 |
|---|---|---|
| 3.1 | 拖一个 wav/mp3/m4a 进 Dragger | 直接转写，不会真的发到 antd 上传后端 |
| 3.2 | 拖一个 25MB+ 的文件 | 前端 Alert "文件超过 25MB 限制"；不发请求 |
| 3.3 | 拖一个 .txt | 前端 Alert "不支持的格式" 或后端 415 |
| 3.4 | 上传 5 秒的 mp3 | 转写完成 < 6 秒（含网络）|

## 4. 会话存储

| # | 操作 | 预期 |
|---|---|---|
| 4.1 | 第一次进 `/conversations` 页 | 左侧空，提示"还没有会话"，下方语音输入可用 |
| 4.2 | 直接录音不点"新建" | 自动建会话（标题截取转写前 30 字），转写消息出现在右侧 |
| 4.3 | 点"新建" → 录音 | 在新建会话下追加消息 |
| 4.4 | 切换不同会话 | 右侧消息气泡按时间正序刷新 |
| 4.5 | 删除当前会话 | 二次确认 → 删除 → 右侧回到空状态 |
| 4.6 | 列表 / 详情 时间戳 | 显示本地时区，updated_at 在追加消息后会变新 |

## 5. 多用户隔离

| # | 操作 | 预期 |
|---|---|---|
| 5.1 | 用户 A 创建一个会话，记下 id | 列表里看到 1 条 |
| 5.2 | 退出 A，注册 B 登录 | B 的列表是空的（看不到 A 的会话）|
| 5.3 | 用 B 直接 `GET /conversations/<A 的 id>`（DevTools fetch）| 404 + `code: conversation_not_found`（**不是 403**，避免暴露存在性）|

## 6. 异常恢复

| # | 操作 | 预期 |
|---|---|---|
| 6.1 | 后端关掉再操作 | 全局 toast 报错，UI 不白屏；前端 token 仍在，重启后端后操作恢复 |
| 6.2 | 把 STEP_API_KEY 改错 | `/stt/transcribe` → 502 + `code: stt_auth`，前端展示"STT 凭证失败或无 ASR 权限" |
| 6.3 | DevTools 把 localStorage 的 access_token 改坏 | 任意受保护操作 → 401 → 自动清登录态 → 跳 `/login` |
| 6.4 | 等 token 过期（默认 7 天，开发可改 jwt_expire_minutes 测试） | 进 `/conversations` 时被 ProtectedRoute 拦截，跳 `/login` |

## 7. 性能口径（参考 PRD 5.1）

| 指标 | 目标 | 怎么测 |
|---|---|---|
| 语音转写延迟 | < 2 秒（句末→文字）| 录 5 秒说一句，停止后到文字出现的体感 |
| 文件上传转写 | < 6 秒（5s 音频）| 上传 5 秒 mp3 看耗时 |
| 页面首屏 | < 1.5 秒 | DevTools Performance |

## 通过定义

- 1.x ~ 6.x 全勾掉，没出现没解释的报错
- DevTools Console 没有红错
- 后端日志 grep `ERROR` 应该是空（warning 可以有，比如登录失败）
